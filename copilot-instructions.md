# Developer Dashboard - Copilot Instructions

## Project Overview
- React dashboard with GitHub activity tracker, task manager, and weather widget
- Uses free public APIs (GitHub API, Open-Meteo API)
- No backend required

## Commands
- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build

## Key Files
- `src/App.jsx` - Main components (GitHubActivity, TaskManager, WeatherWidget)
- `src/index.css` - Styling with CSS variables
- `index.html` - Entry HTML

## Features
1. **GitHub Activity**: Enter username to fetch profile and recent events
2. **Task Manager**: Add/delete tasks, filter by status, localStorage persistence
3. **Weather**: Search city, displays temp/conditions/wind/humidity
